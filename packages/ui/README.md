# @leanlog/ui

Shared UI component library for Leanlog.

## Local development

From repo root:

```bash
pnpm install
pnpm dev:ui
```

Run Storybook:

```bash
pnpm storybook
```

Build:

```bash
pnpm --filter @leanlog/ui build
```
