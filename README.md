# leanlog

Leanlog is a mobile-first nutrition tracker for meals, calories, and macros.

## Stack

- pnpm workspaces (monorepo)
- React + TypeScript
- Vite
- Storybook
- Tailwind CSS
- ESLint + Prettier + Husky

## Packages

- `apps/web` — main app
- `packages/ui` — shared UI components + Storybook

## Local development

```bash
pnpm install
pnpm dev
```

Useful commands:

```bash
pnpm storybook
pnpm lint
pnpm typecheck
pnpm build
```
